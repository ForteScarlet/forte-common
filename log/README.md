# common-logger

日志模块

配置文件: `nekolog[.yml|.properties]`

```properties
# 是否开启language
nekolog.enableLanguage=true
# 是否开启颜色
nekolog.enableColor=true
# 使用的日志等级
nekolog.level=INFO
# 开启输出堆栈信息
nekolog.enableStack=true
```

```yaml
nekolog: 
    # 是否开启language
  enableLanguage: true
    # 是否开启颜色
  enableColor: true
    # 使用的日志等级
  level: INFO
    # 开启输出堆栈信息
  enableStack: true
```
